Příkazy pro vytvoření virtuálního prostředí a instalace Flask:
```python
python -m venv udelej_to                 
.\udelej_to\Scripts\activate
pip install flask
```

Příkazy pro vytvoření databáze:
```python
python create_db.py
```


Příkazy pro spuštění Flasku:
```python
flask --app udelej_to --debug run 
```
